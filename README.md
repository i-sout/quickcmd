# quickcmd
简化命令输入，再也不用记那么多命令拉。

# 安装
* 安装[fzf](https://github.com/junegunn/fzf)
* `git clone https://github.com/i-sout/quickcmd.git ~/.quickcmd`
* 在`~/.bashrc`或`~/.zshrc`或其他终端配置文件中，添加:
  * `[[ -s "~/.quickcmd/quickcmd.sh" ]] && source "~/.quickcmd/quickcmd.sh"`

# 使用
* `qc -h`
* `qc -l`
* `qc`
